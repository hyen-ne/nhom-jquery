# BÀI THỰC HÀNH JQUERY

## Các hiệu ứng đã sử dụng

1. Slick Carousel (Trang chủ - index.html)  
→ Sử dụng plugin jQuery để tạo slider cho banner, giúp hình ảnh tự động chuyển động.

2. Hiệu ứng hover sản phẩm (Trang chủ - index.html)  
→ Khi người dùng rê chuột vào sản phẩm, sản phẩm phóng to nhẹ và thay đổi độ mờ.

3. Hiệu ứng scroll fade-in (Trang chủ - index.html)  
→ Khi người dùng cuộn trang, các phần nội dung (sale, sản phẩm, blog) xuất hiện dần với hiệu ứng fade-in.
